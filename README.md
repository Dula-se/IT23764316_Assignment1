## 📌 Project Description

This project evaluates the accuracy of converting **Singlish (Sinhala written using English letters in chat-style format)** into **Sinhala Unicode text** using an online transliteration tool.

The system tested:
👉 https://www.pixelssuite.com/chat-translator

The evaluation is conducted using **automated testing with Playwright**, focusing only on identifying cases where the system fails to produce correct Sinhala output.

---

## 🎯 Objective

The main objectives of this assignment are:

* Identify **50 negative test cases** where the system fails to correctly transliterate Singlish input
* Ensure coverage of **all 24 Singlish input types** specified in the assignment
* Automate test execution using Playwright
* Record results in an Excel sheet for analysis

## ▶️ How to Run the Project

### Step 1: Install dependencies

```
py -m pip install playwright openpyxl
```

### Step 2: Install Playwright browsers

```
py -m playwright install
```

### Step 3: Run the automation script

```
py "IT23764316 test_automation.py" --excel "TestCases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --type-delay-ms 80 --slow-mo-ms 200 --save-every 1 --keep-open
```

## 📊 Test Case Information

* Total Test Cases: **50**
* Test Type: **Negative (Failure scenarios)**
* Test Case IDs: **Neg_0001 to Neg_0050**

Each test case includes:

* Input (Singlish text)
* Expected Output (Correct Sinhala)
* Actual Output (Generated by system)
* Status (PASS / FAIL)
* Singlish Input Type
* Evidence / Rationale

---

## ⚠️ Important Notes

* Only **FAIL cases** are considered valid for this assignment
* Test cases are designed to **highlight weaknesses in transliteration accuracy**
* The system is expected to produce incorrect outputs for these cases
* Automation results are automatically recorded in the Excel file
